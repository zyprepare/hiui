## Props
| 参数           | 说明                         | 类型               | 可选值                | 默认值    |
| ------------- | ---------------------------- | ----------------- | -------------------- | -------- |
| children         | 文本内容 |  string  |    |    |
| numberOfLines    | 超出行数显示省略号 |  number  |  非必填  |  1  |
| maxTextCount     | 最大文字展示数量 |  number  |  非必填  |    |
| TooltipProps     | tooltip 组件 Props |    |  非必填  |    |
